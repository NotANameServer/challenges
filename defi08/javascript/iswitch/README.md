# Connect 4

> Implementation of the **Connect 4** game for web browsers.

## 0. Prerequisites

* Latest version of `npm`

## 1. Installation

* `npm install`

## 2. Setup

1. `npm run serve`
2. Open `http://localhost:8080/` in your favorite browser.


## 3. Usage

* See [index.ts](./index.ts).

## 4. Build

1. `npm run build`
2. Open `./dist/index.html` in your favorite browser.

## 5. Clean

* `npm run clean`
